2. Attribute 绑定

v-bind专门的简写语法: v-bind:class 改为 :class

3. 事件监听

v-on 监听 DOM 事件, 简写语法：v-on:click 写为 @click

increment 加入methods 选项声明，@click="increment"

this 来访问组件实例

4.表单绑定

使用 v-bind 和 v-on 来在表单的输入元素上创建双向绑定

v-model 为上述语法糖，v-model 会将被绑定的值与 <input> 的值自动同步，不必再使用事件处理函数

v-model 不仅支持文本输入框，也支持诸如多选框、单选框、下拉框之类的输入类型。
